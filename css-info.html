CSS FUTURE

1. Basic Syntax

Example:

h1 { font-size: 30px; }
h1 → selector
{ } → Declaration
font-size: → Property
30px; → Value

2. Styling Methods

Inline Styling (Highest Priority)

<h1 style="property:value;"> ... </h1>

Internal Styling

<head>
<style>
    h1 {
        property: value;
    }
</style>
</head>

External Styling (Lowest Priority)

HTML File:

<head>
<link rel="stylesheet" href="style.css">
</head>

CSS File:

h1 {
    property: value;
}

3. HTML Refresher

Element: <p>this is a paragraph</p>
Tag: <p></p>
Content: this is a paragraph
Attribute: <tag attribute="value">Text Content</tag>
DOM: Document Object Model
Comments: /* Text */

4. Selectors

Element Selector

h1 { }

Universal Selector

* { }

ID Selector

#ID_name { } (Only for one specific element)

Class Selector

.Class_name { } (For group of elements)

Group Selector

h1, h2, p, #ID_name, .Class_name { }

Descendant Selector

#abc p i { } Targets <i> inside <p> inside #abc

5. Properties

Color Properties

Color: color: green;
RGB: color: rgb(255, 0, 124);
Hex: color: #ff006b;
RGBA: color: rgba(255, 0, 124, 0.73);

Size Properties

Height: height: 50px;
Width: width: 50px;

Background Properties

Background Color: background-color: red;
Background Image: background-image: url(...);
Background Size: background-size: cover;
Background Repeat: background-repeat: no-repeat;
Background Position: background-position: center;

Text Properties

Text Align: text-align: center;
Text Decoration: text-decoration: underline;
Text Transform: text-transform: uppercase;
Line Height: line-height: 20px;

Font Properties

Font Size: font-size: 25px;
Font Weight: font-weight: bold;
Font Style: font-style: italic;
Font Family: font-family: Arial, Helvetica, sans-serif;

6. Shorthand Properties

Color

color: value;

Background

background: color image repeat attachment position;

Visibility

visibility: hidden/visible;
